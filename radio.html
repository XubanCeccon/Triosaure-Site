Maintenant que nous en savons plus sur chaque moyen de communication,nous allons calculer la portée théorique de chacun de nos composants. Pour cela, nous nous sommes basés sur les données de puissance, sensibilité et le gain pour calculer la portée estimée de chacun des émetteurs récepteurs. Pour les calculs de la portée nous nous sommes basés sur la formule de FRIIS :

Pr = Pe−Pce+Ge−Pt+Gr−Pcr (en dB)

avec Pt = -147.56 +20log(f) + 20log(D)

Ainsi on obtient: 20log(D) = Pe - Pce + Ge + 147.56 - 20log(f) + Gr - Pcr -Pr

Ensuite il nous suffit de résoudre l’équation. les résultats obtenus sont ceux dans le tableau ci-dessus. Par sécurité, nous avons cherché les valeurs de portée types de chacune des voies de communication afin de s’assurer de l’exactitude de nos calculs.
